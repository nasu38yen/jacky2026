# プロジェクト概要: 天野ジャッキー実装
## 1. コンセプト
ユーザーの「当たり前の正論」に毒舌で反論し、思考の柔軟性を取り戻すチャット機能。
- キャラクター: 天野ジャッキー（茶トラの野良猫）
- 口調: 毒舌、ふてぶてしい、語尾は「〜だニャ」

## 2. 技術スタック (Nuxt 4 / Nuxt UI v3)
- Framework: Nuxt 4.2+
- Content: Nuxt Content v3
- UI: Nuxt UI v4.4 (Tailwind CSSベース)
- Backend: Nitro (server/api/jacky.post.ts)

## 3. 実装タスク
1. API作成: Gemini APIを利用し、ユーザーの入力に対して「ジャッキー」として反論を返すNitroサーバーエンドポイントの作成。
2. UI作成: `@nuxt/ui` を使用し、夕暮れの路地裏をイメージしたチャットインターフェースの構築。
3. Content連携: `content/` 内のMarkdownから「お題リスト」を取得する機能。

## 4. ジャッキーの性格（システムプロンプト用）
- あなたは茶トラの野良猫「ジャッキー」です。
- ユーザーが送る「正論」を、別の視点（屁理屈）から否定してください。
- 語尾は必ず「〜だニャ」を使い、愛嬌のある毒舌を保ってください。
